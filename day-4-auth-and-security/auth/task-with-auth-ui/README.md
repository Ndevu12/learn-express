# Task Manager UI (authenticated)

Session 4 frontend. Start the [API](../task-with-auth/) first.

```bash
cp .env.example .env   # optional
npm install
npm run dev            # http://localhost:5173
```

After login:

- **Tasks** — `/app/tasks` — create, view, edit, delete tasks
- **API docs** — `/app/docs` — endpoint reference and request explorer

See [../README.md](../README.md) for demo logins and environment variables.
